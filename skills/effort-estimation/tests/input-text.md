# 需求：内部员工学习平台（MVP）

## 背景

公司需要一套面向内部员工的学习平台，支持课程发布、员工选课、学习进度跟踪与管理员后台运营。首期为 MVP，不涉及复杂的推荐算法。

## 功能清单

### 1. 用户登录

- 支持企业微信扫码登录
- 首次登录自动同步企业通讯录信息（姓名、部门、工号）
- 支持登录态 7 天自动续期

### 2. 课程中心

- 课程列表浏览，按分类 / 难度筛选
- 课程详情页：标题、讲师、章节列表、简介、封面
- 支持收藏、点赞

### 3. 学习进度

- 视频章节学习进度条（秒级精度）
- 完成一个章节后自动解锁下一章
- 个人学习仪表盘（总学习时长、完成课程数、在学课程）

### 4. 在线考试

- 课程末尾提交选择题试卷（含单选、多选、判断）
- 自动判分、生成结业证书（PDF）
- 成绩不合格支持 3 次补考

### 5. 管理员后台

- 课程 CRUD（含章节、视频上传到对象存储）
- 学员学习数据看板（按部门、按课程聚合）
- 导出学习报表（Excel）

### 6. 消息通知

- 新课程发布站内信推送
- 考试通过 / 未通过邮件通知
- 管理员可手动群发公告

## 约束

- 技术栈：后端 Spring Boot + MySQL + Redis，前端 Vue 3
- 视频存储对接阿里云 OSS
- 上线时间：期望 2 个月内
